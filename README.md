## ✨ V3 시스템 주요 특징

### 🔥 핵심 기능
- **키움 동적 손절 시스템**: 실제 운용되던 예리한 손절 로직 완전 포팅
- **VWAP 리버전 전략**: 편차 기반 평균회귀 전략 고도화
- **멀티브로커 지원**: 시뮬레이션, IBKR 라이브/페이퍼 지원
- **롱 전용 안전 거래**: 공매도 리스크 완전 차단
- **실시간 모니터링**: 멀티 종목 동시 모니터링

- 동적 손절선 자동 설정
- 트레일링 스톡 수익 보호
- 실시간 리스크 모니터링
